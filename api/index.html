Organicity APIs

Public APIs

Asset Discovery

The Asset Discovery API facilitates exploration and inspection of available assets.

Asset Federation

The Asset Federation API provides access to asset creation and update services of the Organicity Platform.

Data Source

The Data Source API provides access to the data stored in Organicity.

Asset Annotation

The Asset Annotation API provides access to asset annotation servives of Organicity.

Permissions

The Permission Component of Organicity Accounts provides inspection, editing and updating of access permissions.

Authentication and Authorisation API

The Authentication and Authorisation API handles the login and the corresponding access rights.

Smartphone Experimentation

The Authentication and Authorisation API handles the login and the corresponding access rights.

Internal APIs

Experimentation Management

The Experimentation Management API provides means to modify the experiments and acquire accounting information regarding experiments

Facility Management

The Facility Management API handles the management of the OragniCity Facility

Community Management

The Community Management API provide interfaces to manage communities, involve participants in experiments and create incentivisation and rewarding policies
